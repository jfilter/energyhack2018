Energyhack 2018 Gewinner

von Johannes Filter und Andreas Corusa, 13.10.2018, Code
Zum Energyhack 2018 präsentieren wir eine künsterlische und spielerische Auseinandersetzung mit einem Datensatz. Um was für Daten könnte es sich handeln? Rate mit. Wir geben dir auch Tipps!




Auflösung

Es handelt sich um den Zulauf zum Klärwerk, die von den Berliner Wasserbetrieben bereitgestellt wurden. Ein Zyklus bildet einen Tag ab und die Messung sind 15-minütig für das gesamte Jahr 2017. Link zum Datensatz.
